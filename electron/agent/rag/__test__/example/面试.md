# 技术面试准备指南

## 数据结构与算法

### 常见数据结构
- **数组与链表**: 数组支持随机访问，时间复杂度O(1)；链表支持高效插入删除，时间复杂度O(1)
- **栈与队列**: 栈是后进先出(LIFO)结构；队列是先进先出(FIFO)结构
- **哈希表**: 通过哈希函数实现O(1)平均时间复杂度的查找
- **树与图**: 二叉搜索树、AVL树、红黑树、B+树等

### 常见算法
- 排序算法：快速排序、归并排序、堆排序
- 搜索算法：二分搜索、BFS、DFS
- 动态规划：背包问题、最长公共子序列
- 贪心算法：活动选择、哈夫曼编码

## 系统设计

### 分布式系统基础
- CAP定理：一致性、可用性、分区容错性三者最多只能同时满足两个
- 负载均衡：轮询、加权轮询、最少连接数
- 缓存策略：LRU、LFU、TTL

### 常见设计题
1. 设计一个URL缩短服务
2. 设计一个消息队列系统
3. 设计一个分布式文件存储系统
4. 设计一个实时聊天应用
